What is included

  • Temporal feature engineering from longitudinal patient history
  • Model calibration and subgroup fairness analysis
  • Cost-sensitive thresholding and queue-based prioritization
  • Workflow simulation and ROI estimation
  • External validation pipeline
  • Interactive Streamlit app with queue export

Quickstart

  • Create a virtual environment and install dependencies from `requirements.txt`
  • Train with `python -m src.train --data-path data/raw/diabetic_data.csv`
  • Evaluate with `python -m src.evaluate --data-path data/raw/diabetic_data.csv`
  • Launch the app with `streamlit run app/streamlit_app.py`
